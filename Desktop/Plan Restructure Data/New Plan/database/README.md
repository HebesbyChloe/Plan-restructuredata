# Database Files

This folder contains all database-related files organized by schema version.

## Structure

### `old_schema/`
Original MySQL schema files (before migration)
- DDL files from original database
- Original table definitions
- Migration reference files

### `new_schema/`
New PostgreSQL schema files
- Standardized table definitions
- Migration scripts
- Schema documentation

### `postgres_inserts/`
PostgreSQL INSERT statements
- Sample data
- Test data
- Seed data scripts

## Files

- `DDL_Database_CRM.sql` - Original MySQL DDL (root folder)
- Migration scripts in `migrations/` folder (root folder)
