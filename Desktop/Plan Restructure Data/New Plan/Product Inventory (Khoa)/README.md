# Product Inventory Department (Khoa)

This folder contains documentation and code related to product and inventory management.

## Tables

- `product` - Product catalog
- `category` - Product categories
- `product_category` - Product-category junction table
- `product_tag` - Product tags
- `stock` - Inventory stock levels
- `product_attribute` - Product attributes
- `material_stock` - Material inventory

## Responsibilities

- Product catalog management
- Inventory tracking
- Stock management
- Product attributes and categories
- Material inventory
