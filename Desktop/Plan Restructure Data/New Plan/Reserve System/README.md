# Reserve System Department

This folder contains documentation and code related to the reserve/backup system.

## Tables

- `reserve_order` - Reserve system orders
- `reserve_product` - Reserve system products
- `reserve_stock` - Reserve system stock
- `reserve_category` - Reserve categories
- `reserve_attribute` - Reserve attributes

## Responsibilities

- Reserve system management
- Backup order processing
- Alternative product catalog
- Separate inventory tracking
