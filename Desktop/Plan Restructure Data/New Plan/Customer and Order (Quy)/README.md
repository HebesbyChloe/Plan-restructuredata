# Customer and Order Department (Quy)

This folder contains documentation and code related to customer and order management.

## Tables

### Customer Management
- `customer` - Customer records
- `customer_batch` - Customer batches
- `customer_lead` - Customer leads
- `pancake_contact` - Pancake contact integration

### Order Management
- `order` - Orders
- `order_detail` - Order details
- `order_line_item` - Order line items
- `payment` - Order payments
- `refund` - Refunds
- `order_return` - Order returns

## Responsibilities

- Customer relationship management
- Order processing
- Payment tracking
- Returns and refunds
- Customer batch management
