# Campaign and Marketing Department

This folder contains documentation and code related to marketing campaigns and promotions.

## Tables

- `campaign` - Marketing campaigns
- `campaign_ads` - Campaign ads junction table
- `campaign_ads_running` - Running ads junction table
- `campaign_target_audience` - Target audiences junction table
- `campaign_collection` - Campaign collections junction table
- `promotion` - Promotions
- `promotion_category` - Promotion categories
- `promotion_product` - Promotion products
- `promotion_attribute` - Promotion attributes

## Responsibilities

- Marketing campaign management
- Promotion management
- Ad tracking
- Audience targeting
- Collection management
